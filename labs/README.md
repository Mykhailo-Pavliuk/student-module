# Опис папок

Ця папка labs містить задачі з програмування, розділені за семестрами навчання.

## Папка Basic
Містить задачі з програмування, які відповідали першому семестру навчання.  
Ці завдання призначені для практичного опрацювання базових навичок кодування.

## Папка OOP
Містить задачі з об’єктно-орієнтованого програмування (ООП), які відповідають другому семестру навчання.  
Папка поступово буде поповнюватися новими задачами відповідно до програми.
